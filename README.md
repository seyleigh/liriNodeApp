# Liri Node App

This a command line app that acts like Siri but uses user input in place of voice command. 

------------------------------------------

### Technologies used:

* fs
* axios
* moment.js
* Spotify API
* Bands in Town API
* OMDB API

[Link to GitHub repo](https://github.com/seyleigh/liriNodeApp)

------------------------------------------

```javascript

node liri.js concert-this

```
